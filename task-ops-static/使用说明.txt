Task Ops 离线版 — 直接使用
============================

1. 用浏览器打开本文件夹里的 index.html
   · 可以双击 index.html，或把 index.html 拖到浏览器窗口里
   · 推荐使用 Chrome / Edge / Safari

2. 无需安装 Node.js，无需联网（界面与数据都在本机）

3. 演示登录
   邮箱：admin@demo.com
   密码：demo123
   （manager@demo.com、member@demo.com 密码相同）

4. 数据保存在当前浏览器的「本地存储」中；清除站点数据会重置演示数据

5. 部署在 https://task-ops-static.vercel.app 时，默认会把数据同步到后端
   https://task-ops-ivory.vercel.app（需在该项目执行数据库迁移并配置 AUTH_SECRET）。
   若你的同步 API 地址不同，可在页面控制台执行：
   localStorage.setItem("workgraph_api_base","https://你的API根地址")
   或设置 window.WORKGRAPH_SYNC_API 后刷新。

6. 若双击后仍是空白页，请右键 index.html → 打开方式 → 选浏览器

7. 若数据无法保存，请换用 Chrome 或 Edge 打开；不要用「无痕/隐私」模式
